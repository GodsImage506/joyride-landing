# Joyride Motorsports — Landing Pages

Pre-approval landing pages for Joyride Motorsports vehicle financing.

## Stack
- Node.js + Express
- Static HTML/CSS/JS
- Hosted on Railway

## Deployment
Auto-deploys on push to main via Railway.

## Usage
Each page is linked from Rick, the Joyride AI lead engagement bot.
Rick sends leads a direct URL matching the vehicle they inquired about via Facebook Messenger.
